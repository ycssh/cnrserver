三龙
一对子+两杠牌+一三头，胡一手可走


1.以上牌型清一色翻倍，放胡必须胡
2.组成
  1）牌局麻将使用由条、饼、万、中、发、白，总计120张牌组成
  2）游戏人数为4人，每人手里13张牌，不能吃牌，可以碰和杠
3.定庄
第一把每人打一次骰子，以骰子点数大的为庄，以后每把定庄规则如下：
  1）有人走，由第一个放弃继续胡牌的人做庄
  2）无人走且只有一个没听牌，由没听牌的陪叫且做庄
  3）无人走且不是只有一个人没听牌，则由第一个胡牌的人或第一个听牌的人或上一把的庄家继续做庄家，优先级由左至右
